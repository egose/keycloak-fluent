# Realm Bootstrap

Use this flow when you need to create a realm from the `master` realm and immediately provision a dedicated realm-admin service account for follow-up automation.

This is a common first step when your infrastructure code needs to hand off from cluster-wide admin access to realm-scoped credentials.
